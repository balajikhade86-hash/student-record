# Student Record Management System in Python

## Project Overview
This is a simple Student Record Management System developed using Python. The application allows users to add, view, and delete student records.

## Features
- Add student details
- View student records
- Delete student records
- Simple menu-driven interface

## Technologies Used
- Python

## How to Run
1. Open the project in VS Code
2. Run the Python file
3. Add, view, or delete student records
4. Exit the program

## Project Outcome
This project helped in understanding Python lists, dictionaries, loops, conditions, and user input handling.
